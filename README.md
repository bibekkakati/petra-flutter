## Petra

Period Tracker app that predicts the next date of the menstruation phase and notifies you of the same.

### Features:

-   Predict Date
-   Track Cycle
-   Delete Cycle
-   View Cycle
-   Notification on Date

### Technology used:

-   Flutter
-   Dart
-   BLoC Pattern
-   SQLite
